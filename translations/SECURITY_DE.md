# Sicherheitsrichtlinie

## Unterstützte Versionen

In diesem Abschnitt kannst du angeben, welche Versionen deines Projekts aktuell mit Sicherheitsupdates unterstützt werden.

| Version | Unterstützt        |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Meldung einer Sicherheitslücke

Wir nehmen die Sicherheit von Zen C sehr ernst. Solltest du eine Sicherheitslücke im Zen C-Compiler, der Standardbibliothek oder den zugehörigen Tools gefunden haben, melde diese bitte wie unten beschrieben.

**Bitte melde Sicherheitslücken nicht über öffentliche GitHub-Issues.**

### So meldest du Sicherheitslücken

Bitte sende uns eine E-Mail an **security@zenc-lang.org**.

Dein Bericht sollte Folgendes enthalten:

- Die Version von Zen C, die du verwendest (`zc --version`).
- Eine Beschreibung der Sicherheitslücke.
- Ein minimales reproduzierbares Beispiel (MRE) oder eine Schritt-für-Schritt-Anleitung zur Reproduktion des Problems.
- Alle relevanten Konfigurations- oder Umgebungsdetails.

Du erhältst innerhalb von 48 Stunden eine Antwort. Wird das Problem bestätigt, stellen wir schnellstmöglich einen Patch bereit.
